## Annotation

```
import java.lang.annotation.;

@Target({ElementType.METHOD, ElementType.TYPE})
@Retention(RetentionPolicy.RUNTIME)
@Documented
public @interface RequestMatrix {

    String value() default "";
}
```

###### 说明

```
@Retention
注解的生存周期，即会保留到哪个阶段
 - SOURCE：源码级别保留，编译后即丢弃。
 - CLASS:编译级别保留，编译后的class文件中存在，在jvm运行时丢弃，这是默认值。
 - RUNTIME： 运行级别保留，编译后的class文件中存在，在jvm运行时保留，可以被反射调用
    
@Target
注解的使用范围，即被描述的注解可以用在哪里
 - TYPE:类，接口或者枚举
 - FIELD:域，包含枚举常量
 - METHOD:方法
 - PARAMETER:参数
 - CONSTRUCTOR:构造方法
 - LOCAL_VARIABLE:局部变量
 - ANNOTATION_TYPE:注解类型
 - PACKAGE:包
 
@Documented
可以被例如javadoc此类的工具文档化，只负责标记，没有成员取值
```

## Aspect

Pointcut 

指那些方法需要被执行"AOP",是由"Pointcut Expression"来描述的

```java
@Aspect
@Component
public class WebParamAspect {

    @Pointcut("@annotation(com.example.demo.RequestMatrix)")
    public void pointCut() {}

    @Before("pointCut()")
    public void doBefore(JoinPoint joinPoint) {
    }

    @Around("pointCut()")
    public Object doAround(ProceedingJoinPoint joinPoint) throws Throwable {
        return joinPoint.proceed();
    }

    @AfterReturning("pointCut()")
    public void doAfterReturning() {

    }
}
```

说明

```
@Aspect
将一个类定义为一个切面类

@Around 
环绕切点,在进入切点前,跟切点后执行

@Before
在切点前执行方法,内容为指定的切点

@After
在切点后,return前执行,
```
