# Vite

脚手架，Rollup打包器

> https://github.com/vitejs/vite/tree/main/packages/create-app

```
# 搭建Vue3
yarn create vite-app <project>

# 搭建React
mkdir <project>
npm init vite-app --template react

# 支持TypeScript
<script lang="ts">
```
