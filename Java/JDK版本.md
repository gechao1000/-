## 发布周期

每年 3 月，9月 （6个月为周期）

每半年发布一个大版本，每个季度发布一个中间特性版本

长期支持版本每 3 年发布一次

#### 现状

```
70% oraclejdk
21% openjdk

jdk8 79%
```
