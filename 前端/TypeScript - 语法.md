TypeScript

>  https://www.tslang.cn/ 

```
# 安装
npm install -g typescript 
或
pacman -S typescript

# 编译
tsc helloworld.ts

# 项目中使用
yarn --init -y
tsc --init
yarn add @types/node --dev
```
