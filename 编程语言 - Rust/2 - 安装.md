### Manjaro 开发环境

```
# 安装 C 编译器
base-devel	（默认已经安装）

# 安装
pacman -S rust

# 配置文件
~/.cargo/config
```

### Windows 开发环境

```
# 安装 C 编译器
Build Tools for Visual Studio 2019

# 安装
scoop install rustup

# 配置文件
D:\Scoop\persist\rustup\.cargo\config
```
