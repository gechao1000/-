## 使用

###### 基础命令

```shell
# 提示
brew 
# 在线搜索软件包
https://formulae.brew.sh/

brew serach xxx
brew install xxx

# 安装Casks
brew cask install xxx 
```

###### 安装 brave浏览器

```
https://brave.com/zh/

brew search brave
brew cask install brave-browser
```
